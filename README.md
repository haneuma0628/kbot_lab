卒研してる学生の毒舌な後輩bot概要
--------
以下参照
[卒研してる学生の毒舌な後輩bot取り扱い説明書](http://hassakutea.com/kbot_lab/ "hoge")

仕様
--------
* 2時間おきにランダムな毒舌を吐きますが、たまにデレます
* 10:10に卒研発表会までの残り日数を16進数で呟きます(発表会終了のため来年度までお待ちください)
* 毎時5分から55分の間10分おきにmentionリストを取得して、最新のmentionに反応します(現在停止中)
